Experimental Chat Interface - Test Page

Development and testing interface for experimental conditions

About This Test Interface

This is a development/testing interface for the experimental chat framework. It allows you to:

  1. Test different experimental conditions
  2. Verify the chat interface loads correctly
  3. Ensure condition-specific styling and behavior works
  4. Debug issues during development

How to Use:

  1. Select a condition from the dropdown below
  2. Click "Load Chat" to initialize the chat interface
  3. Interact with the AI assistant to test functionality
  4. Verify that the condition's styling and behavior match your configuration
  5. Check different conditions to ensure all work correctly

Configuration:

Experimental conditions are defined in experimental_conditions.json. Each condition can specify:

  • Bot name and icon
  • Visual styling (colors, header visibility)
  • System prompts (AI behavior)
  • Model parameters (temperature, token limits)

For details on configuring conditions, see: docs/EXPERIMENTAL_CONDITIONS_GUIDE.md
